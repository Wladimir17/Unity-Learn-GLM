## Глава 1. Введение в синтаксис и типы данных

### 1.1. Теория
C# — это строго типизированный, объектно-ориентированный язык. Программа состоит из классов, а точка входа — метод `Main`.

**Основные типы данных:**
*   `int` (целые числа): 1, 50, -20
*   `double` (вещественные числа): 3.14, 10.5
*   `string` (строка): "Hello World"
*   `bool` (логический): true, false
*   `var` (явное выведение типа): компилятор сам определяет тип.

### 1.2. Пример кода
```csharp
using System;

namespace MyApp
{
    class Program
    {
        static void Main(string[] args)
        {
            // Объявление переменных
            string name = "Алекс";
            int age = 25;
            double height = 1.75;
            bool isStudent = true;

            // Вывод на экран (интерполяция строк)
            Console.WriteLine($"Имя: {name}, Возраст: {age}, Рост: {height}");

            // Ввод данных
            Console.Write("Введите ваш возраст: ");
            string input = Console.ReadLine();
            int userAge = int.Parse(input); // Преобразование строки в число

            Console.WriteLine($"Через 5 лет вам будет: {userAge + 5}");
        }
    }
}
```

## Глава 2. Условные операторы и циклы

### 2.1. Теория
Управляющие конструкции определяют поток выполнения программы.

*   **Условия:** `if`, `else if`, `else`, тернарный оператор `? :`.
*   **Циклы:** `for` (когда известно количество итераций), `while` (пока условие истинно), `do-while` (хотя бы раз).

### 2.2. Пример кода
```csharp
using System;

class Program
{
    static void Main()
    {
        // Пример условия
        int number = 10;

        if (number % 2 == 0)
        {
            Console.WriteLine("Число четное");
        }
        else
        {
            Console.WriteLine("Число нечетное");
        }

        // Тернарный оператор
        string result = (number > 0) ? "Положительное" : "Отрицательное или ноль";

        // Пример цикла for
        Console.WriteLine("Обратный отсчет:");
        for (int i = 5; i >= 1; i--)
        {
            Console.WriteLine(i);
        }
        Console.WriteLine("Пуск!");

        // Пример цикла while для проверки пароля
        string password = "12345";
        string userInput;
        
        do
        {
            Console.Write("Введите пароль: ");
            userInput = Console.ReadLine();
        } while (userInput != password);
        
        Console.WriteLine("Доступ разрешен.");
    }
}
```

## Глава 3. Массивы и коллекции

### 3.1. Теория
Для хранения множества элементов используются массивы (фиксированный размер) и коллекции (динамический размер).

*   **Массив:** `int[] nums = new int[5];`
*   **Список:** `List<int> list = new List<int>();` (находится в пространстве имен `System.Collections.Generic`).

### 3.2. Пример кода
```csharp
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        // Массив
        int[] numbers = { 10, 20, 30, 40, 50 };
        
        Console.WriteLine("Элементы массива:");
        for (int i = 0; i < numbers.Length; i++)
        {
            Console.WriteLine(numbers[i]);
        }

        // Список (динамический)
        List<string> tasks = new List<string>();
        tasks.Add("Помыть посуду");
        tasks.Add("Сделать уроки");

        // Удаление элемента
        tasks.Remove("Помыть посуду");

        Console.WriteLine("\nСписок задач:");
        foreach (var task in tasks)
        {
            Console.WriteLine("- " + task);
        }
    }
}
```

## Глава 4. Методы (Функции)

### 4.1. Теория
Методы позволяют разбить код на логические блоки для повторного использования.
Синтаксис: `Модификатор ТипВозврата ИмяМетода(Параметры) { ... }`

### 4.2. Пример кода
```csharp
using System;

class Program
{
    // Метод, который возвращает int
    static int Sum(int a, int b)
    {
        return a + b;
    }

    // Метод, который ничего не возвращает (void)
    static void PrintHello(string name)
    {
        Console.WriteLine($"Привет, {name}!");
    }

    // Метод с перегрузкой (одно имя, разные параметры)
    static double Sum(double a, double b)
    {
        return a + b;
    }

    static void Main()
    {
        int result = Sum(5, 10);
        Console.WriteLine($"Сумма 5 и 10 = {result}");

        PrintHello("Иван");
        
        double res2 = Sum(5.5, 4.5);
        Console.WriteLine($"Сумма 5.5 и 4.5 = {res2}");
    }
}
```

## Глава 5. Основы ООП (Классы и Объекты)

### 5.1. Теория
ООП строится на трех китах: Инкапсуляция, Наследование, Полиморфизм. В этой главе мы рассмотрим основы классов.
*   **Класс:** чертеж (шаблон).
*   **Объект:** экземпляр класса.
*   **Свойства:** доступ к данным (get/set).
*   **Конструктор:** метод для инициализации объекта при создании.

### 5.2. Пример кода
```csharp
using System;

public class Person
{
    // Поля (private для инкапсуляции)
    private string _name;
    private int _age;

    // Свойство для имени
    public string Name
    {
        get { return _name; }
        set { _name = value; }
    }

    // Автосвойство для возраста
    public int Age { get; set; }

    // Конструктор
    public Person(string name, int age)
    {
        Name = name;
        Age = age;
    }

    // Метод класса
    public void Introduce()
    {
        Console.WriteLine($"Меня зовут {Name}, мне {Age} лет.");
    }
}

class Program
{
    static void Main()
    {
        // Создание объекта
        Person person1 = new Person("Анна", 28);
        person1.Introduce();

        Person person2 = new Person("Борис", 35);
        person2.Introduce();
        
        // Изменение свойства
        person1.Age = 29;
        Console.WriteLine($"{person1.Name} теперь {person1.Age}");
    }
}
```

## Глава 6. Наследование и Полиморфизм

### 6.1. Теория
*   **Наследование:** Позволяет создать новый класс на основе существующего (`class Dog : Animal`).
*   **Полиморфизм:** Возможность использовать методы с одинаковыми именами для разных типов (ключевые слова `virtual`, `override`).

### 6.2. Пример кода
```csharp
using System;

// Базовый класс
public class Animal
{
    public string Name { get; set; }

    public Animal(string name)
    {
        Name = name;
    }

    // Virtual означает, что метод можно переопределить
    public virtual void MakeSound()
    {
        Console.WriteLine($"{Name} издает звук.");
    }
}

// Дочерний класс
public class Dog : Animal
{
    public Dog(string name) : base(name) { }

    // Override — переопределение метода
    public override void MakeSound()
    {
        Console.WriteLine($"{Name} говорит: Гав-гав!");
    }
}

public class Cat : Animal
{
    public Cat(string name) : base(name) { }

    public override void MakeSound()
    {
        Console.WriteLine($"{Name} говорит: Мяу!");
    }
}

class Program
{
    static void Main()
    {
        Animal myDog = new Dog("Шарик");
        Animal myCat = new Cat("Мурка");

        // Полиморфизм: мы вызываем один и тот же метод, но поведение разное
        myDog.MakeSound();
        myCat.MakeSound();
    }
}
```
